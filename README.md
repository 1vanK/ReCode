# ReCode

Простая утилита для изменения кодировки множества файлов.

Порядок действий:

1. Перетащите необходимые файлы (папки) в список.
2. Выберите исходную кодировку.
3. Выберите требуемую кодировку.
4. Нажмите кнопку "Старт".

Примечания:

* Если на список перетащить папки, то в него (список) добавятся файлы из этих папок и всех вложенных папок.
* Клавишей DEL из списка можно удалить лишние файлы.
* В файлы с расширениями ".bak" сохраняются резервные копии перекодируемых файлов.
* Для компиляции запустите Build.bat.
